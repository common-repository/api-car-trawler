== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `api-for-car-trawler` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[api-for-cartrawler-form redirect='false' ]` in your page

== Frequently Asked Questions ==

= Can my customer really book a car anywhere in the world with this booking engine? =

Absolutely, give it a try yourself!

= Are my visitors personal and payment details secure? =

Absolutely. Although 100% security can never be guaranteed, every search query loads a secure page that displays a list of rental cars. The page and the security is provided by CarTrawler.
